## 🏗️ The 4 Capstone Projects

<div align="center">
<img src="https://user-images.githubusercontent.com/74038190/212257467-871d32b7-e401-42e8-a166-fcfd7baa4c6b.gif" width="100" />
</div>

### 🐳 Project 1 — Containerized Web App + CI/CD
> **Stack:** Python Flask/Node.js · Docker · GitHub Actions · AWS EC2  
> **Duration:** 5–7 days  
Build and auto-deploy a web app end-to-end with Docker containers and a full GitHub Actions pipeline pushing to AWS.

---

### ☸️ Project 2 — Kubernetes Microservices Deployment
> **Stack:** Docker · Kubernetes (minikube/EKS) · Helm · Prometheus · Grafana  
> **Duration:** 7–10 days  
Deploy 3 real microservices with Helm charts, auto-scaling, Ingress, and a full observability stack.

---

### 🏗️ Project 3 — Infrastructure as Code with Terraform
> **Stack:** Terraform · AWS (VPC, EC2, RDS, ECS) · Remote State  
> **Duration:** 5–7 days  
Provision an entire cloud infrastructure — VPC, EC2, RDS, ECS — using only Terraform code. No ClickOps.

---

### 🚀 Project 4 — Full End-to-End DevOps Platform *(Advanced)*
> **Stack:** Terraform · EKS · ArgoCD · GitOps · Prometheus · Loki · Cert-Manager · Vault  
> **Duration:** 2 weeks  
Build a production-grade platform from scratch — the kind companies actually run in production.

## 💡 Golden Rules from the Document

<div align="center">

> 🔨 **Build, break, fix** — you learn more from debugging than tutorials  
> 📝 **Document everything** — write READMEs, draw architecture diagrams  
> 🌐 **Go public** — push all your projects to GitHub from Day 1  
> 👥 **Join communities** — you'll learn 3x faster with others  
> 🏅 **Certifications after knowledge** — understand first, certify second  
> 🛠️ **The fundamentals never change** — Linux, networking, and Git always matter
